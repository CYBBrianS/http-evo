What is HTTP/2?

HTTP/2 is a major revision of the HTTP protocol released in 2015 by the Internet Engineering Task Force (IETF). It was designed to improve the performance of web browsing by addressing the limitations of HTTP/1.1. HTTP/2 was now using multiplexing, meaning it can send multiple streams of data down the same connection. This decreases latency and eliminates the "head-of-line" blocking problem that existed in earlier versions. It also introduced header compression, which compresses the data sent in headers to make it smaller and faster.

Key Features of HTTP/2

  • Multiplexing: Multiple requests/responses in one connection
  • Header Compression: Reduces overhead for faster performance
  • Server Push: Allows server to send resources before browser asks
  • Stream Prioritization: Optimizes the order of resource loading
